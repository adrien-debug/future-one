Mix énergétique recommandé
~88% Gaz · ~12% Solaire PPA (+ BESS en stockage transitoire, hors mix)
Phase 1 (200 MW) : 100% gaz Kahramaa via raccordement HTB + groupes électrogènes diesel pour secours.
Phase 2–3 : introduction PPA solaire 100–200 MW depuis Al Kharsaah ou nouveau solar farm captif Umm Alhoul.
Phase 4–5 : ~12% renouvelable atteint · BESS Tesla/Fluence pour lissage transitoire (~30 sec, pas une source) · objectif PUE <1,2.
Long terme : évaluer SMR si projet GCC concrétisé (horizon 2032+).
Note : BESS = stockage de transition entre coupure réseau et démarrage des groupes électrogènes. Capacité indicative ~80–160 MWh, pas comptabilisée dans le mix de production.
